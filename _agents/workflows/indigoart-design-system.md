# Indigoart (富士家 藍染サイト) デザインシステム & コンポーネント集
※別チャット（別ページ）へ移行した際、Up-cycleページで確立したデザインルールを引き継ぐための定義書。

## 1. タイポグラフィと段落ち防止（絶対ルール）
*   **スマホ段落ち防止クラス**: `<br class="u-br-pc" />`
    *   PCでは改行。スマホ（全幅）では改行を無効化し、自然な折り返し（`text-wrap: pretty;` / `word-break: auto-phrase;`）に委ねる。
*   **絶対分割禁止ブロック**: `<span style="display:inline-block;">お申込み</span>` 等
    *   ボタン内やCTAなど、絶対に単語の途中で切れてはならない文字列に使用。

## 2. レイアウトの基本幅とコンテナ
*   **PC幅の制御**: テキスト主体のセクションは必ず `max-width: 800px;`（または `700px`）等の制限をかけ、横いっぱいに文字が広がる「ギチギチの壁」を禁止する。
*   **基本セクション余白**: `.lp-sub-section` (padding: 60px 20px) などを活用し、セクション間の呼吸を担保する。

## 3. 見出しデザイン
*   **共通の意匠見出し**: `.lp-sub-section-title`
    *   中央揃えで、上部に装飾線（菱形や縦線）をあしらった和モダンな高級見出し。単なる文字の拡大やボーダーベタ塗りは禁止。
    *   ※フォントは明朝体（`font-family: serif;`）を基調とする。

## 4. アイコン＋カードレイアウト（直感性の重視）
機能や強みを羅列する際は、**絶対に箇条書きのテキストベタ打ちにしないこと**。
*   `.lp-sub-feature-grid` のようなグリッドレイアウト（`display: grid` または `flex`）を利用。
*   カード（`.lp-sub-feature-card`）内に以下を配置：
    1. 直感的なSVGアイコン（白抜き・円形など和を乱さないデザイン）
    2. 見出し
    3. 説明文
*   背景色はやたらと濃い色に変えず、ベースは明色（既存のテクスチャ等）を維持し、カードの方に影（`box-shadow`）等をつけることで立体感・高級感を出す。

## 5. 絶対遵守事項（Master SKILL.md PHASE 13連動）
*   無駄な英語（UVスラグ名など）をデザインに入れない。
*   AI画像を無断で本番に貼らない（`/assets/`の既存画像のみ使う）。
*   ハンバーガーメニュー等、既存JS機能の破壊を防ぐ（完了前に必ず `browser_subagent` で実機クリック検証する）。
